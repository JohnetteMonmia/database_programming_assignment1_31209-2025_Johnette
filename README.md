#LearnersLoop- Peer Tutoring Management System
Database Programming Assignment

## Business Idea 
LearnersLoop is a tutoring management system designed to help university students who need additional academic support get the hekp they need. It utilizes student tutors who are accademically equiped to provide support to their fellow students. The system manages student registrations, tutor profiles, course information, tutoring sessions, and student feedback in a centralized database.

## Business Problems
students struggle with understanding certain courses and need additional accademic support outside regular lectures.There is no centralized system to help students find tutors, schedule sessions, or evaluate tutoring quality.
## Proposed Solution 
LearnersLoop provides a database-driven platform where students can search for tutors, book tutoring sessions, and submit feedback. Administrators can use the database to monitor tutor performance, track student participation, and improve academic support services.

## Database Schema
The database consists of five related tables:
* Students
* Courses
* Tutors
* TutoringSessions
* Feedback
The tables are linked using primary keys and foreign keys to maintain data integrity.

## Common Table Expressions (CTEs)
The project demonstrates:
* Simple CTE
* Multiple CTEs
* Recursive CTE
* Aggregation CTE
* JOIN CTE

## Analysis of Findings
The SQL queries show that peer tutoring can be monitored effectively by tracking tutor ratings, tutoring sessions, and student participation. The analysis also helps identify high-demand courses and top-performing tutors.
